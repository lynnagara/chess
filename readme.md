# Chess

2 player chess in the browser using &lt;canvas&gt;



### Javascript


Run tests:

* npm install
* karma start


### ScalaJS

Prerequisites:

* sbt 0.13.7 or newer
* scala 2.11.6 or newer

SBT commands:

* sbt
* run / ~run
* test / ~test
* reload => reloads build file
* fastOptJS => rebuild js
